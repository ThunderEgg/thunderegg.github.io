Detailed Description

template<int D>
class ThunderEgg::GMG::CycleBuilder< D >

Builder for building GMG cycles.

User will provide Operator,Smoother,Restrictor, and Interpolator objects for each level.

addFinestLevel has to be called first, then addIntermediateLevel (if there are any), and finally addCoarsestLevel. Then getCycle can be called to get the completed Cycle. If these are called in the wrong order, an exception will be thrown.

Template Parameters
Dthe number of cartesian dimensions
