Detailed Description

template<size_t D>
class ThunderEgg::MPIGhostFiller< D >

Parallell ghostfiller implimented with MPI.

There are two private functions that have to be overridden derived classes. fillGhostCellsForNbrPatch, and fillGhostCellsForLocalPatch

Template Parameters
Dthe number of Cartesian dimensions
