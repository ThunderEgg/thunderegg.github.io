Detailed Description

template<size_t D>
class ThunderEgg::Schur::InterfaceDomain< D >

Represents the Schur compliment domain of the problem.

This class mainly manages a set of interface that makes up the Schur compliment system. It is responsible for setting up the indexing of the interfaces, which is used in the rest of the ThunderEgg library.

Dthe number of Cartesian dimensions
