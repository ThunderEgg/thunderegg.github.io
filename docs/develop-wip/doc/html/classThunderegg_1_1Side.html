Detailed Description

template<size_t D>
class Thunderegg::Side< D >

An enum-style class that represents the sides of a patch.

The sides of the patch are named in the following way:

Orthogonal to axis Lower on axis Higher on axis
x-axis west east
y-axis south north
z-axis bottom top
Template Parameters
Dthe number of cartesian dimensions
