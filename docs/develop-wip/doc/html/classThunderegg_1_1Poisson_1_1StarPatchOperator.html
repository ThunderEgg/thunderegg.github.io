Detailed Description

template<size_t D>
class Thunderegg::Poisson::StarPatchOperator< D >

Implements 2nd order laplacian operator.

Supports both Dirichlet and Neumann boundary conditions

Template Parameters
Dthe number of Cartesian dimensions
