Detailed Description

template<size_t D>
class Thunderegg::GMG::Cycle< D >

Base class for cycles. Includes functions for preparing vectors for finer and coarser levels, and a function to run an iteration of smoothing on a level. Derived cycle classes need to implement the visit function.
