Detailed Description

template<size_t D>
class ThunderEgg::VarPoisson::StarPatchOperator< D >

Implements a variable coefficient Laplacian f=Div[h*Grad[u]].

h is a cell-centered coefficient

Template Parameters
Dthe number of Cartesian dimensions
