Detailed Description

template<size_t D>
class Thunderegg::GMG::CycleFactory< D >

Factory for producing GMG cycles.

User has to provide functions that return Restrictors, Interpolators, Smoothers, and Operators for the GMG Levels.

Dthe number of cartesian dimensions
