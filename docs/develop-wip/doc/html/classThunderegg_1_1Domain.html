Detailed Description

template<size_t D>
class Thunderegg::Domain< D >

Represents the domain of the problem.

This class mainly manages a set of patches that makes up the domain. It is responsible for setting up the indexing of the domains, which is used in the rest of the Thunderegg library.

Template Parameters
Dthe number of Cartesian dimensions
